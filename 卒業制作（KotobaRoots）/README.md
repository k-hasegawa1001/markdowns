# [KotobaRoots]

## 📖 プロジェクト概要

このリポジトリは、[文化や歴史から言語を学ぶ学習アプリ] を実現するために開発されたWebアプリケーションのソースコードです。

---

## ⚠️ 開発者・引き継ぎ担当の方へ（必ず読んでください）

このドキュメントは、**学校の演習環境（AWS Academy / Learner Lab）**で実際に動作することを確認済みです。
ネット上の記事とは手順が異なる場合（特にIPアドレスや鍵の扱いなど）がありますが、基本的にはこのドキュメントの手順に従ってください。

このプロジェクトは、授業で習った一般的な構成（`flask run` だけで動かす簡易的なもの）とは異なり、**実際の現場での運用を想定した「本格的な構成」**を採用しています。

そのため、初めて見るファイルや設定があるかもしれません。
「なぜそんな面倒なことをするの？」「バックエンドとフロントエンドって何？」という疑問を解消するために、**専門用語を使わずに解説したドキュメント**を用意しました。

コードを触る前に、まずは以下のドキュメント（`docs/`フォルダ内）を**上から順に**読んでください。

## 📂 ディレクトリ構成（全体像）

このプロジェクトは、大きく分けて「画面（フロント）」と「処理（バック）」に分かれています。

```text
root/
├── frontend/          # ユーザーが見る画面 (React/HTML/CSSなど)
│                      # -> レストランで言う「メニュー表・内装」
│
├── backend/           # 裏側の処理・データベース (Flask/Djangoなど)
│                      # -> レストランで言う「厨房・シェフ」
│
├── docs/              # 解説ドキュメント
│                      # -> このプロジェクトの「攻略本」
│
└── .github/workflows/ # 自動デプロイの設定ファイル (CI/CD)
```
